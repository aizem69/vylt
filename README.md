Getting Started

1. **Clone the repo:**
```bash
git clone https://github.com/aizem69/vylt.git
cd vylt
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the app:**
```bash
python app.py
```

4. **Open in browser:**  
`http://localhost:5000`
